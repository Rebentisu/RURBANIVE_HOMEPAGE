# RUE#2 Ecosystem and Biodiversity Restoration Homepage

An interactive homepage developed as part of the RURBANIVE project for RUE#2: Ecosystem and Biodiversity Restoration.

Lead Partners: Agricultural University of Athens (AUA) and IMERYS

## Overview

This homepage presents a seasonal clock that visualizes the progression of the year in real time. At its center, a dynamic image illustrates representative vegetation states of Milos, changing according to the current season.

The purpose of the homepage is to provide an intuitive and engaging entry point for exploring time, seasonality, and landscape transitions within the context of ecosystem and biodiversity restoration.

## Features

* Real-time seasonal clock
  Displays the current date, time, season, and day of the year through a circular annual visualization

* Season-based vegetation display
  The central image updates dynamically according to the season (Winter, Spring, Summer, Autumn)

* Representative Milos vegetation imagery
  The displayed imagery is conceptual and illustrative, showing seasonal vegetation variation in Milos rather than real-time environmental measurements

* Interactive navigation toolbar
  Animated sidebar menu linking to the main sections of the platform

* Responsive visual design
  Designed to adapt across screen sizes with smooth transitions and interactive elements

## Technologies

* HTML5
* CSS3
* Vanilla JavaScript


## Concept Note

The vegetation imagery displayed in this homepage is illustrative and not based on real-time satellite or NDVI data. It is intended to communicate seasonal landscape changes in a simplified and accessible way.
